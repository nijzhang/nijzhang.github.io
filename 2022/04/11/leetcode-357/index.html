leetcode-357 统计各位都不同的数字个数

这是leetcode-357 统计各位都不同的数字个数, 难度为【中等】。

给你一个整数n,统计并返回各位数字都不同的数字x的个数，其中 0<= x < 10 n

示例2：

提示：

  • 0 <= n <= 8

分析

最容易想到的办法是逐个枚举判断，但是这样做必定TLE，再深入思考一下，可以发现，这是《概率论》里面提到的乘法原理,其实很简单，比如：
三位数：100~999，由于首位不可以为0，所以百位只有9种选择，十位去除百位已选的一个数字,避免重复还剩下9种选择，个位得去除百位、十位已经用的，剩下8种选择，故三位数数字不重复一共有9*9*8;
二位数：同理 9*9;
一位数： 10;
故 n=3时， 返回9*9*8 + 9*9 + 10

代码

  • 时间复杂度: O(n)
  • 空间复杂度: O(1)
