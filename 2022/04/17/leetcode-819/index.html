leetcode-819. 最常见单词

发表于 2022-04-17 分类于 leetcode每日一题 阅读次数：
本文字数： 1.8k 阅读时长 ≈ 3 分钟

这是leetcode上的819. 最常见的单词， 难度为【简单】。

给定一个段落 (paragraph) 和一个禁用单词列表 (banned)。返回出现次数最多，同时不在禁用列表中的单词。

题目保证至少有一个词不在禁用列表中，而且答案唯一。

禁用列表中的单词用小写字母表示，不含标点符号。段落中的单词不区分大小写。答案都是小写字母。

示例:

提示:

  • 1 <= 段落长度 <= 1000
  • 0 <= 禁用单词个数 <= 100
  • 1 <= 禁用单词长度 <= 10
  • 答案唯一， 都为小写字母， 即便在paragraph里面是大写，答案都是小写；
  • paragraph只包含字母、空格、下列标点符号!?',:.
  • 单词里只包含字母、不会出现省略号或者其他标点符号

简单模拟

  • 时间复杂度: O(n)
  • 空间复杂度: O(n)
